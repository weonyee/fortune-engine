# 사주 해석 DB 구조 문서

## 개요

사주명리학(四柱命理學) 기반의 천간(天干) / 지지(地支) 해석 데이터베이스이다.
전통 명리 원전을 인용하고, 현대적 해석을 확장하여 구성하였다.

---

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 만세력 계산기 웹 페이지 (HTML 마크업) |
| `style.css` | 전체 CSS (반응형 포함) |
| `js/engine.mjs` | 사주 해석 엔진 — 데이터 + 순수 계산 로직 |
| `js/main.mjs` | 엔진 import + 렌더링 + calculate() |
| `js/ui.js` | 폼, 저장, 드로어, 모달 DOM 이벤트 |
| `cheongan_db.py` | 천간(天干) 10개 기본 성향 + 일간 상세 해석 + 관계(십성/합/충) |
| `jiji_db.py` | 지지(地支) 12개 기본 성향 + 일지 상세 해석 + 관계(육합/삼합/방합/충/형/파/해) |

## 1. cheongan_db.py — 천간 DB

### 1-1. CHEONGAN (천간 10개 기본 성향)

```python
CHEONGAN = {
    "甲": {
        "한글": str,      # "갑"
        "음양": str,      # "양" | "음"
        "오행": str,      # "목" | "화" | "토" | "금" | "수"
        "상징": str,      # "큰 나무, 대들보"
        "성향": str,      # 기본 성향 요약
        "키워드": list,   # ["리더십", "정직", ...]
    },
    # ... 乙, 丙, 丁, 戊, 己, 庚, 辛, 壬, 癸
}
```

### 1-2. ILGAN_DETAIL (일간 상세 해석)

일간(日干)에 해당 천간이 있을 때 기준의 심층 분석 DB이다.

```python
ILGAN_DETAIL = {
    "甲": {
        "원전": {
            "적천수": str,   # 한문 원문
            "해석": str,     # 원문 해석
        },
        "특성_총평": str,        # 1. 특성 총합 및 총평
        "표현_행동양식": str,    # 2. 표현 / 행동 양식
        "교류방식": str,         # 3. 사람들과 교류하는 방식
        "강점": list[str],       # 4-1. 강점 (5개)
        "약점": list[str],       # 4-2. 약점 (5개)
        "개선방안": str,         # 4-3. 개선 방안
        "추천직업": list[dict],  # 5. 추천 직업 (분야/역할/이유)
        "종합요약": {
            "강점_살리기": str,  # 6-1. 강점 살리는 방안
            "약점_보완하기": str, # 6-2. 약점 보완하는 방안
        },
    },
    # ... 乙 ~ 癸 (10개 전체)
}
```

### 1-3. 천간 관계 DB

| 변수 | 설명 |
|------|------|
| `CHEONGAN_HAP` | 천간합(天干合) 5쌍 — 결과 오행 + 설명 |
| `CHEONGAN_CHUNG` | 천간충(天干沖) 4쌍 — 충돌 설명 |

### 1-4. 해석 함수

| 함수 | 설명 |
|------|------|
| `_get_oheng_relation(me, other)` | 오행 기준 관계 산출 |
| `_get_sipsung(ilgan, target)` | 십성(十星) 산출 |
| `interpret_ilgan(ilgan_char)` | 일간 기본 성향 출력 |
| `interpret_ilgan_detail(ilgan_char)` | 일간 심층 분석 출력 (원전 포함) |
| `interpret_relation(ilgan, target)` | 일간 vs 타 천간 관계 해석 |

## 2. jiji_db.py — 지지 DB

### 2-1. JIJI (지지 12개 기본 성향)

```python
JIJI = {
    "子": {
        "한글": str,      # "자"
        "음양": str,      # "양" | "음"
        "오행": str,      # "목" | "화" | "토" | "금" | "수"
        "동물": str,      # "쥐"
        "시간": str,      # "23:00~01:00"
        "월": str,        # "11월 (음력)"
        "상징": str,      # "지혜, 시작, 어둠 속의 생명력"
        "성향": str,      # 기본 성향 요약
        "키워드": list,   # ["지혜", "민첩", ...]
        "지장간": list,   # ["壬", "癸"]
    },
    # ... 丑 ~ 亥 (12개 전체)
}
```

### 2-2. ILJI_DETAIL (일지 상세 해석)

일지(日支)에 해당 지지가 있을 때 기준의 심층 분석 DB이다.

```python
ILJI_DETAIL = {
    "子": {
        "원전": {
            "연해자평": str,  # 한문 원문
            "삼명통회": str,  # 한문 원문
            "해석": str,      # 원문 해석
        },
        "특성_총평": str,        # 1. 특성 총합 및 총평
        "표현_행동양식": str,    # 2. 표현 / 행동 양식
        "교류방식": str,         # 3. 사람들과 교류하는 방식
        "강점": list[str],       # 4-1. 강점 (5개)
        "약점": list[str],       # 4-2. 약점 (5개)
        "개선방안": str,         # 4-3. 개선 방안
        "추천직업": list[dict],  # 5. 추천 직업 (분야/역할/이유)
        "종합요약": {
            "강점_살리기": str,  # 6-1. 강점 살리는 방안
            "약점_보완하기": str, # 6-2. 약점 보완하는 방안
        },
    },
    # ... 丑 ~ 亥 (12개 전체)
}
```

### 2-3. 지지 관계 DB

| 변수 | 설명 |
|------|------|
| `JIJI_YUKHAP` | 육합(六合) 6쌍 — 결과 오행 + 설명 |
| `JIJI_SAMHAP` | 삼합(三合) 4조 — 결과 국(局) + 설명 |
| `JIJI_BANGHAP` | 방합(方合) 4조 — 결과 국(局) + 설명 |
| `JIJI_CHUNG` | 충(沖) 6쌍 — 충돌 설명 |
| `JIJI_HYUNG` | 형(刑) — 삼형 + 자형 포함 |
| `JIJI_PA` | 파(破) 6쌍 — 파괴 설명 |
| `JIJI_HAE` | 해(害/穿) 6쌍 — 해침 설명 |

### 2-4. 해석 함수

| 함수 | 설명 |
|------|------|
| `interpret_jiji(jiji_char)` | 지지 기본 성향 출력 |
| `interpret_ilji_detail(jiji_char)` | 일지 심층 분석 출력 (원전 포함) |
| `interpret_jiji_relation(a, b)` | 두 지지 간 관계 해석 (합/충/형/파/해) |
| `check_samhap(jiji_list)` | 삼합/방합 완성 여부 확인 |
| `check_jahyung(jiji_list)` | 자형(自刑) 확인 |

### 주의사항

- 원전 원문은 판본에 따라 글자가 일부 다를 수 있음 (경도본, 임철초 주석본 등)
- 현대적 해석(행동양식, 직업 추천 등)은 원전의 상(象)을 바탕으로 **확장 해석**한 것
- 명리학 유파에 따라 해석이 다를 수 있으므로, 실 서비스 적용 시 전문가 감수 권장
