# Fortune Engine

사주명리학 기반의 만세력 계산 및 운세 해석 엔진

## 소개

전통 사주명리학(四柱命理學)의 이론을 데이터베이스화하고, 만세력 계산을 웹에서 수행할 수 있도록 구현한 프로젝트입니다.

- 생년월일시를 입력하면 사주팔자(연주·월주·일주·시주)를 자동 계산
- 천간(天干) 10개, 지지(地支) 12개의 상세 해석 DB를 내장
- 적천수(滴天髓), 연해자평(淵海子平), 삼명통회(三命通會) 등 전통 원전 인용

## 주요 기능

### 만세력 계산기 (`index.html`)

| 기능 | 설명 |
|------|------|
| 사주팔자 계산 | 양력 생년월일시 → 연주·월주·일주·시주 산출 |
| 십성 분석 | 일간 기준 각 주의 십성(비견, 식신, 편재 등) 표시 |
| 지장간 표시 | 각 지지에 내포된 천간(지장간) 표시 |
| 12운성 | 일간 기준 각 지지의 12운성 산출 |
| 시간 보정 | 지역 경도 기반 진태양시 보정 지원 |
| 로컬 저장 | localStorage를 이용한 만세력 저장/불러오기 |

### 천간 DB (`cheongan_db.py`)

천간 10개(甲~癸) 각각에 대해:

- 기본 성향 (음양, 오행, 상징, 키워드)
- 일간 상세 해석 6개 항목
  1. 특성 총합 및 총평
  2. 표현 / 행동 양식
  3. 사람들과 교류하는 방식
  4. 강점, 약점, 개선 방안
  5. 추천 직업과 이유
  6. 종합 요약
- 천간 관계 (십성, 천간합 5쌍, 천간충 4쌍)
- 적천수(滴天髓) 원문 인용 및 해석

### 지지 DB (`jiji_db.py`)

지지 12개(子~亥) 각각에 대해:

- 기본 성향 (음양, 오행, 동물, 시간, 지장간)
- 일지 상세 해석 6개 항목 (천간 DB와 동일 구조)
- 지지 관계 (육합, 삼합, 방합, 충, 형, 파, 해)
- 연해자평(淵海子平) + 삼명통회(三命通會) 원문 인용 및 해석

## 원전 출처

| 문헌 | 저자 | 활용 |
|------|------|------|
| 적천수(滴天髓) | 경도(京圖), 임철초(任鐵樵) 주석 | 천간론 10간 원문 |
| 연해자평(淵海子平) | 서대승(徐大升) | 지지 상(象) 정의 |
| 삼명통회(三命通會) | 만민영(萬民英) | 논십이지(論十二支) |
| 자평진전(子平眞詮) | 심효첨(沈孝瞻) | 형충회합론 참조 |

원문은 ctext.org, 古诗文网, 太极书馆, Wikisource에서 교차 검증하였습니다.

## 기술 스택

- **만세력 계산**: [@fullstackfamily/manseryeok](https://github.com/urstory/manseryeok-js) (고영창 진짜만세력 기반)
- **프론트엔드**: Vanilla JS, HTML/CSS (Pretendard 폰트)
- **해석 DB**: Python (데이터 구조화)
- **저장**: localStorage
